# An interactive Bookstore application using Python-tkinter

- using sqlite as the database
- tkinter module for GUI
- code is segregated into frontend and backend for GUI and logic clarity
- Users can 
	- add
	- delete
	- modify
	- search
	- view all records
- Books are stored with
	- Title
	- Year
	- author
	- Date

# How to run

- python3 frontend.py
